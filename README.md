# GECK Extender

The GECK Extender is an [NVSE](https://github.com/xNVSE/NVSE) plugin that 
enhances the [GECK](https://geckwiki.com/index.php?title=Garden_of_Eden_Creation_Kit) by fixing various bugs and adding new tools.
